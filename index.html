Аналог игры “Doodle Jump” "Doodle Jump" Game Clone

Doodle Jump - первая разработанная мной программа, имеющая графический интерфейс. Правила игры предельно просты: играя за упавшую с неба звезду, наша задача помочь ей попась обратно на небо, поднимаясь прыжками по платформам. Игра бесконечна, что позволяет совершенстовать свой навык подъема. Если персонаж упал и игра закончилась, то ее всегда можно перезапустить по нажатию кнопки в левом верхнем углу. Управление звездой производится клавишами A(влево) и D(вправо). Игра расчитана на детей от 3 лет, благодаря ей они смогут развить мелкую моторику, а также внимательность и реакцию. Doodle Jump is the first program I developed with a graphical interface. The game rules are very simple: playing as a falling star, our goal is to help it get back to the sky by jumping on platforms. The game is endless, allowing you to improve your climbing skills. If the character falls and the game ends, it can always be restarted by pressing the button in the top left corner. The star is controlled using the A (left) and D (right) keys. The game is designed for children from 3 years old, as it helps them develop fine motor skills, attention, and reaction time.

Информационная система для магазина одежды Information System for Clothing Store

ИС для магазина одежды - программа для кассового аппарата, которая ведет собственный учет товаров, а также собирает статистику о продажах, создает отчетность о выручке и прибыли. Этот проект подключен к базе данных PostgreSQL, что позволяет хранить данные о товарах, их количестве и продажах. В программе реализована система оплаты, но условная, таким образом деньги, конечно, списываться не будут, но если товара нет на складе(в базе), то купить его будет невозможно. Внутри программы есть функция поступления, позволяющая не прибегая к базе данных вносить в нее данные о поступлении определенного количества товара на склад, а также корректировать цены. Также, внутри программы есть возможность печати отчетов о прибыли. Если покупатель отказался от покупки товара, то продавец может “отменить” чек, удалив все позиции товаров. The Information System for a clothing store is a program for a cash register that keeps track of inventory, collects sales statistics, and generates reports on revenue and profit. This project is connected to a PostgreSQL database, allowing it to store data about products, their quantity, and sales. The program has a payment system, but it's a mock one, meaning no actual money is deducted. However, if an item is out of stock (in the database), it cannot be purchased. The program includes an incoming stock feature, which allows entering the data about the arrival of a certain quantity of goods into the warehouse without accessing the database, as well as adjusting prices. Additionally, the program provides the ability to print profit reports. If a customer cancels a purchase, the seller can "cancel" the receipt by deleting all product entries. The system also allows for product categorization, making it easier for employees to locate and manage inventory.

Телеграмм бот с расписанием Telegram Bot with Schedule

Телеграмм бот с расписанием - бот, который создавался, как аналог ВятГУ бот. Для этого проекта было создано 2 кода: сам бот и код-парсинг Excel файлов с расписанием с занесением всех данных в базу данных PostgreSQL, откуда бот при пользовании берет информацию для распределения ее по группам. Отличительной особеностью нашего бота от бота ВятГУ является наличие функции уведомлений о парах за неделю, за день и за час. The Telegram bot with a schedule is a bot that was created as an alternative to the VyatSU bot. For this project, two codes were developed: the bot itself and the code for parsing Excel files with the schedule and inserting all the data into a PostgreSQL database, from which the bot retrieves the information to distribute it across groups. The distinguishing feature of our bot compared to the VyatSU bot is the presence of notification functions about upcoming classes for the week, the day, and the hour.
